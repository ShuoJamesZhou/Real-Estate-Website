# appRealEstate website

The site is served by the Node app in `server/`.

## Source of truth

- **Source (editable)**: `server/src/public-src/`
- **Build output (QA/prod-like)**: `server/src/public/` (generated by `cd server && npm run build`)

## Ports

- **Prod**: `3006`
- **QA**: `5006`
- **Local dev**: `5106`

## Run local dev

From the project folder:

```bash
../startLocalDev.sh
```
